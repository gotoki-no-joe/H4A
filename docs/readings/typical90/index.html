# ★4 (14問 2残し)

  • 1. Yokan Party 　ACコード
    出題 　解説 　貪欲法、二分探索

    出題 　解説 　DFS、木の直径

    　出題 　解説 　状態 DP による高速化
  • 12. Red Painting 　ACコード
    出題 　解説 　連結判定は Union-Find
    Unboxed版のUnionFindを改造するともっと速くできる
  • 26. Independent Set on a Tree 　ACコード
    出題 　解説 　二部グラフの性質を使おう
    素朴に葉から塗り分けたノードを集めてconcatするとTLEした。concatを避けるためだけにRoseTreeを持ち出した大袈裟なACコード。上のACコードは、情報を集約する計算をStateモナドに包んで、根からの深さ優先走査の間を持ちまわせるようにした。
  • 28. Cluttered Paper 　ACコード
    出題 　解説 　領域加算は二次元いもす法
  • 34. There are few types of elements 　ACコード
    出題 　解説 　単調性を利用した尺取り法
    数字ごとのカウンタとしてIntMapを直接使うとTLEする。 同じロジックをTypeScriptで書いたらACなのだが。 カウンタとしてmutable vectorを用いて、添え字を圧縮する「座標変換」を行ったものが上のAC。
  • 42. Multiple of 9 　ACコード
    出題 　解説 　9 の倍数の性質
  • 43. Maze Challenge with Lack of Sleep 　[ACコード]
    出題 　解説 　拡張 BFS／拡張ダイクストラ
  • 58. Original Calculator 　ACコード
    出題 　解説 　周期性を考える
    べき乗を使った別解ACコード
    解説のやり方はmutableなので抵抗あるが、時間計算量も空間計算量も圧倒的だ。
  • 63. Monochromatic Subgrid 　ACコード
    出題 　解説 　変な制約に着目する 状態数が少ない変量を全探索
  • 70. Plant Planning 　ACコード

  • 72. Loop Railway Plan 　[ACコード]

    出題 　解説 　DP復元
